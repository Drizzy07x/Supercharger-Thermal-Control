Choose how your Pixel 9 handles heat and performance with a clean WebUI and simple profile switching.

## Overview

Supercharger Thermal Control is an addon module designed to work alongside **Supercharger Pixel 9 Pro Series**.

It gives you a simple way to switch between thermal profiles depending on how you use your device:

- **Balanced** — best everyday balance of temperature, comfort, and performance
- **Gaming** — better sustained performance for longer gaming sessions
- **Charge Cool** — cooler behavior while charging or during longer warm sessions

## Features

- Clean and simple WebUI
- Fast profile switching
- Restart only when needed
- Built for Pixel 9 series
- Compatible with KernelSU setups using Hybrid Mount

## Default behavior

- Default profile on install: **Balanced**
- Restart is only required after switching profiles
- The addon is designed to stay lightweight and stable

## Requirements

- Pixel 9 series supported target
- Root environment supported by your setup
- KernelSU + Hybrid Mount recommended for this release path

## Installation

1. Install the module
2. Reboot
3. Open the WebUI
4. Choose your preferred mode
5. Reboot if the UI asks for it

## Notes

- Best compatibility is expected on stock-based Pixel 9 builds
- Compatibility may vary on custom ROMs with different thermal frameworks
- This addon is meant to complement the main Supercharger module, not replace it
